# Build Resources

## Icon

Place your application icon here:

- `icon.ico` - Windows icon (256x256 recommended)
- `icon.png` - macOS/Linux icon (512x512 recommended)

You can generate icons from PNG using online tools like:

- https://convertico.com/
- https://icoconvert.com/

## Requirements

- ICO format for Windows
- At least 256x256 resolution
- Include multiple sizes: 16x16, 32x32, 48x48, 128x128, 256x256
